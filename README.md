# React TypeScript Vite 프로젝트

이 프로젝트는 React, TypeScript, Vite를 사용하여 구축된 모던 웹 애플리케이션입니다.  
TailwindCSS가 스타일링에 사용되었으며, Bun을 패키지 매니저로 활용합니다.

## 시작하기

### 필수 조건

Bun v1.0.0+ 설치

### 설치

```bash
bun install
```

### 개발 서버 실행

```bash
bun run dev
```

### 빌드

```bash
bun run build
```

### 프리뷰

```bash
bun run preview
```

### 타입 검사

```bash
bun run type-check
```

### 린팅 검사

```bash
bun run lint
```

### 포멧팅

```bash
# 포멧팅 체크
bun run format:check
# 포멧팅 적용
bun run format:fix
```

### 클린업

```bash
bun clearn
```
